# Code of Conduct

## Our Standards

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) community standards.

In short: be respectful, constructive, and professional in all interactions — in Issues, Discussions, and pull requests.

## Enforcement

- For general questions or concerns: open a thread in [GitHub Discussions](https://github.com/KirKruglov/claude-skills-kit/discussions) and mention `@KirKruglov`.
- For private incidents: use GitHub's **Report abuse** feature, which notifies the maintainer privately.

## Scope

This Code of Conduct applies within all project spaces (Issues, Discussions, pull requests) and when an individual represents the project in public spaces.
