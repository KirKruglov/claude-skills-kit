# Как установить feature-guide

Пошаговая инструкция по добавлению **feature-guide** в ваш Claude.

---

## Что понадобится

- Аккаунт Claude (бесплатный или платный)
- Доступ к одному из: Claude.ai Projects, Claude Cowork или Claude Code

---

## Вариант 1: Claude.ai (Projects)

1. Откройте [claude.ai](https://claude.ai) и войдите в аккаунт
2. Нажмите **Projects** в левой панели
3. Создайте новый проект или откройте существующий
4. Нажмите на иконку **⚙ Project settings**
5. Найдите раздел **Custom instructions**
6. Скопируйте всё содержимое файла `SKILL.md` и вставьте в поле инструкций
7. Нажмите **Save**
8. Начните новый диалог внутри проекта — скил активен

---

## Вариант 2: Claude Cowork (приложение для рабочего стола)

1. Откройте приложение Claude
2. Перейдите в **Settings → Skills** (или в папку скилов)
3. Создайте новую папку с именем `feature-guide`
4. Скопируйте `SKILL.md` в эту папку
5. Перезапустите Claude или откройте новую сессию — скил доступен

---

## Вариант 3: Claude Code (CLI)

1. В директории проекта создайте папку для скилов (если её нет):
   ```
   mkdir -p .claude/skills/feature-guide
   ```
2. Скопируйте `SKILL.md` в `.claude/skills/feature-guide/`
3. Скил подключится автоматически в следующей сессии Claude Code

---

## Проверка установки

Начните новый диалог и попробуйте одну из фраз:

- «Есть ли у Claude память?»
- «Что такое Projects?»
- «Умеет ли Claude создавать файлы?»

Если Claude ответит структурированной карточкой (название, доступность, как использовать, ограничения, статус, вердикт) — установка прошла успешно.

---

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Скил не срабатывает | Убедитесь, что скопировали **весь** файл SKILL.md, включая YAML-заголовок |
| Устаревшая информация в карточках | Убедитесь, что web search включён в вашей сессии Claude |
| Неправильный язык | Скил поддерживает русский и английский — попробуйте фразу на другом языке |
