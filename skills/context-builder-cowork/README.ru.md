# Context Builder Cowork — скил Claude

> Создаёт структурированный файл `project-context.md` для Claude Cowork через пошаговое интервью — готов к загрузке и добавлению в проект.

---

## Что делает

Context Builder Cowork проводит структурированное интервью и собирает полный файл `project-context.md`, который Claude использует как постоянный контекст проекта в Cowork. Два режима:

- **Полный режим** — 15 вопросов в 9 блоках: основное, суть проекта, текущая цель, структура, участники, ограничения, пути к данным, открытые вопросы, дополнительные разделы
- **Быстрый режим** — только первые 3 блока (8 вопросов); остальные разделы заполняются плейсхолдерами `[уточнить]`

---

## Как работает

1. Запускаешь скил фразой «создай контекст проекта»
2. Claude задаёт вопросы блоками — отвечаешь в своём темпе
3. Claude собирает файл `project-context.md` из твоих ответов
4. Скачиваешь файл и добавляешь в папку проекта в Cowork

Если у тебя уже есть вся информация — можно передать её одним сообщением, Claude соберёт файл без интервью.

---

## Быстрый старт

Открой диалог с Claude и напиши:

- «Создай контекст проекта»
- «Новый проект контекст для аналитического проекта»
- «Быстрый контекст — нужен минимальный набор»
- «Сформируй контекстный файл для Cowork»

---

## Режимы

| Режим | Триггер | Блоки |
|-------|---------|-------|
| Полный (`new`) | «создай контекст проекта» | Все 9 блоков, 15 вопросов |
| Быстрый | «быстрый контекст» | Блоки 1–3, 8 вопросов; остальное → плейсхолдеры |

---

## Структура выходного файла

```
project-context.md
├── Основное (тип, статус, даты)
├── Project Overview
├── Vision
├── Цель текущего этапа
├── Структура проекта
├── Участники и зависимости
├── Ограничения
├── Структура данных (пути input/output)
├── Открытые вопросы
└── Лог изменений
```

---

## Требования

- Аккаунт Claude (бесплатный или платный)
- Claude Cowork или Claude.ai (любой интерфейс)

---

## Что скил НЕ делает

- Не читает существующие файлы — контекст строится только из твоих ответов
- Не домысливает факты — неизвестные поля становятся плейсхолдерами `[уточнить]`
- Не требует Cowork для запуска — работает в любом диалоге с Claude
