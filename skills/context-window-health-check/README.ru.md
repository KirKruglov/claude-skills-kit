# Проверка состояния контекста

Скил для Claude, который оценивает состояние текущей сессии и говорит простым языком — продолжать работу, сделать handoff или начать заново. Без счётчиков токенов, без кода.

## Обзор

Длинные сессии Claude могут незаметно деградировать: Claude начинает терять ранние детали, повторяться или давать противоречивые ответы. Этот скил обнаруживает предупредительные сигналы и за секунды даёт понятный статус с конкретным действием — для нетехнических пользователей, которые не хотят разбираться в токенах и контекстных окнах.

В отличие от CLI-инструментов (например, счётчика токенов в Claude Code), скил работает полностью внутри интерфейса чата.

## Требования

Файлы не нужны. Скил анализирует текущий разговор напрямую.

## Как использовать

1. Напишите один из триггеров ниже (или опишите симптом)
2. Скил анализирует сигналы сессии: количество сообщений, разнообразие тем, упомянутые симптомы
3. Вы получаете: статус (🟢 / 🟡 / 🔴), объяснение простым языком и одну конкретную рекомендацию

## Примеры

**Пример 1 — Явный запрос**
> «Проверь мой контекст — мы работаем уже долго.»

Ответ: `🟡 Caution — Сессия охватывает несколько тем и идёт около 25 сообщений. Claude может начать терять ранние детали. Рекомендация: сделай handoff перед следующим крупным блоком задач. → Используй session-handoff-composer.`

**Пример 2 — Описание симптома**
> «Ты как будто забыл, о чём мы договорились раньше по структуре проекта.»

Ответ: `🔴 Critical — Ты сообщил, что Claude теряет ранний контекст. Это сильный сигнал: сессия превысила надёжный объём рабочей памяти. Рекомендация: начни новую сессию. → Сначала используй session-handoff-composer, чтобы сохранить ключевой контекст.`

## Триггеры

| Русский | English |
|---------|---------|
| `проверь мой контекст` | `check my context` |
| `оценка контекста` | `context health check` |
| `контекст заканчивается?` | `is my context running out` |
| `мне нужна новая сессия или можно продолжать` | `should I start a new session` |

## Расшифровка статусов

| Статус | Значение | Рекомендация |
|--------|----------|--------------|
| 🟢 Healthy | Сессия в норме | Продолжай работу |
| 🟡 Caution | Сессия длинная или разветвлённая | Сделай handoff перед следующей крупной задачей |
| 🔴 Critical | Высокий риск потери контекста | Начни новую сессию; сначала используй `session-handoff-composer` |

## Связанные скилы

- **session-handoff-composer** — создание структурированного handoff-документа при переходе в новую сессию
