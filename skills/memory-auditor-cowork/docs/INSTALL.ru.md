# Как установить memory-auditor-cowork

Пошаговая инструкция по добавлению **memory-auditor-cowork** в ваш Claude Cowork.

---

## Что понадобится

- Аккаунт Claude (бесплатный или платный)
- Десктопное приложение Claude Cowork

---

## Вариант 1: Claude Cowork (приложение для рабочего стола)

1. Откройте приложение Claude
2. Перейдите в **Settings > Skills** (или в папку скилов)
3. Создайте новую папку с именем `memory-auditor-cowork`
4. Скопируйте `SKILL.md` в эту папку
5. Перезапустите Claude или откройте новую сессию — скил доступен

---

## Вариант 2: Claude Code (CLI)

1. В директории проекта создайте папку для скилов (если её нет):
   ```
   mkdir -p .claude/skills/memory-auditor-cowork
   ```
2. Скопируйте `SKILL.md` в `.claude/skills/memory-auditor-cowork/`
3. Скил подключится автоматически в следующей сессии Claude Code

---

## Проверка установки

Начните новую сессию Cowork и попробуйте одну из фраз:

- "аудит памяти"
- "проверь память"
- "почисти память"

Если Claude ответит обзором слоёв памяти (auto-memory, CLAUDE.md, User Preferences, Project Instructions) — установка прошла успешно.

---

## Важно

Этот скил предназначен специально для **сред Cowork**. Он работает с файловой памятью (auto-memory, CLAUDE.md), а не с нативной памятью Claude.ai. Если вы используете веб-интерфейс Claude.ai, установите [memory-auditor-chat](../memory-auditor-chat/).

---

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Скил не срабатывает | Убедитесь, что скопировали **весь** файл SKILL.md, включая YAML-заголовок |
| auto-memory не найден | Директория `.auto-memory/` должна существовать и содержать хотя бы файл `MEMORY.md` |
| Неправильный язык | Скил поддерживает русский и английский — попробуйте фразу на другом языке |
| Скил предлагает использовать memory_user_edits | Возможно, установлена не та версия — этот скил использует файловые инструменты (Read/Write/Edit), а не memory_user_edits |
